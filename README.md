# LINE Rocket Bot Demo

ระบบนี้เป็นเวอร์ชันทดสอบแบบแต้มจำลองเท่านั้น  
ไม่มีเงินจริง ไม่มีฝากถอน ไม่มีจ่ายเงินจริง

## ติดตั้ง

```bash
pip install -r requirements.txt
```

## ตั้งค่า

คัดลอก `.env.example` เป็น `.env`

```bash
cp .env.example .env
```

แล้วใส่ค่าจริง:

```env
LINE_CHANNEL_ACCESS_TOKEN=...
LINE_CHANNEL_SECRET=...
ADMIN_USER_IDS=Uxxxx,Uyyyy
BACKOFFICE_GROUP_ID=Cxxxx
```

## รันบอท

```bash
python app.py
```

เปิด ngrok:

```bash
ngrok http 5000
```

นำ URL ไปใส่ใน LINE Developers:

```text
https://xxxx.ngrok-free.app/callback
```

## คำสั่งเทส

### เปิดรอบ

```text
เปิด ชัยเมฆา
```

ผลลัพธ์:

```text
ชัยเมฆา

ช่าง ⛔️

🚀🚀🚀🚀🚀
```

### ตั้งราคาช่าง

```text
ราคาช่าง 330-360
```

### โพสต์รายการ

```text
+5ถ850
```

ระบบจะคำนวณเป็น 335-365 จากราคาช่าง 330-360

### ตอบติด

ให้คนที่ต้องการติด กด Reply ข้อความ `+5ถ850` แล้วพิมพ์:

```text
ต
```

หรือ

```text
ติด
```

### เจ้าของโพสต์ยืนยัน

เจ้าของโพสต์กด Reply ข้อความเดิม แล้วพิมพ์:

```text
ต
```

หรือ

```text
ติด
```

เมื่อยืนยันครบ ระบบจะส่ง Flex ไปห้องหลังบ้าน

### ปิดรอบ

```text
ปิด
```

ผลลัพธ์:

```text
❌❌❌❌ ปิด ❌❌❌❌

3 2 1 ไป๊!! 🚀🚀🚀

ชัยเมฆา

⛔หลังปิดไม่ติดทุกกรณี⛔
```

### ดูแต้มจำลอง

```text
เครดิต
```

หรือ

```text
แต้ม
```

## หมายเหตุสำคัญ

- ข้อมูลเก็บใน memory ถ้าปิดโปรแกรมข้อมูลจะหาย
- ถ้าจะใช้จริงสำหรับระบบแต้ม/สมาชิก ควรเปลี่ยนเป็นฐานข้อมูล เช่น PostgreSQL, MySQL หรือ Firebase
- ควรเพิ่มระบบ log, transaction, permission, และระบบกันคำสั่งซ้ำ
